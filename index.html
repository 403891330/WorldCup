参与详情

1、活动开始时间为北京时间2018年6月13 00:00开始时间，活动结束时间为2018年世界杯 1/4决赛结束。

2、(前1000名购买者免手续费)购买每注竞猜将收取5%的手续费，作为平台日常运作资金。

3、总共分为3场比赛分别为(小组赛32进16，1/8决赛，1/4决赛)，每场比赛玩家可进行购买，每一注选择三支球队，并分别指定冠、亚、季军，最低起购价为一注，上不封顶，目前平台只支持ETH为唯一购买凭证。

开奖及中奖规则

1、将在2018世界杯结束后第二天公布中奖名单，并将在结束后48小时内将资金自动打到中奖用户购买的电子钱包地址。

2、总奖金池平台将收取5%的手续费，作为平台日常运作资金。

总奖金池 *0.05 = 分配奖金金额

3、一等奖：3个全竞猜对；(分配奖金金额 *60%) / 一等奖总注数 = 一等奖每注单价。

4、二等奖：竞猜对2个(分配奖金金额 *30%) / 二等奖总注数 = 二等奖每注单价。

5、三等奖：竞猜对1个(分配奖金金额 *10%) / 三等奖总注数 = 三等奖每注单价。

例如: 中奖用户所购买中奖总注数* 用户本次中奖每注单价 = 用户所得奖金

50(注) * 1.2 ETH (一等奖每注单价) = 60 ETH
